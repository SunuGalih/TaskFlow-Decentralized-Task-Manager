# TaskFlow: Decentralized Task Manager

TaskFlow is a lightweight, on-chain Task Management smart contract built on the **Stellar Soroban** network. It allows users to store, manage, and track tasks directly on the blockchain, ensuring data persistence and transparency.

## 🚀 Features

* **Create Task**: Add new tasks with a unique, randomly generated ID.
* **List Tasks**: Fetch the entire collection of tasks stored in the contract instance.
* **Toggle Status**: Mark tasks as completed or pending with a single call.
* **Delete Task**: Clean up your list by removing tasks using their unique ID.
* **No-Std Compliance**: Optimized for the WASM environment on Stellar.

## 🛠 Tech Stack

* **Language**: Rust
* **Platform**: Stellar Soroban
* **SDK**: `soroban-sdk`
